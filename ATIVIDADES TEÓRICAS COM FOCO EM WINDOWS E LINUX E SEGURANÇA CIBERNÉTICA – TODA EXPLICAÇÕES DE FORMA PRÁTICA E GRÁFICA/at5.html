Modelos de Usuários e Permissões

O que é gerenciamento de usuários e permissões?

É o controle que o sistema operacional faz sobre quem pode acessar o quê no computador. Isso ajuda a proteger arquivos, programas e o próprio sistema, evitando que usuários não autorizados façam mudanças ou vejam informações privadas.

Como funciona no Windows (sistema fechado – NTFS)

O Windows usa o sistema de arquivos NTFS, que permite definir permissões para usuários específicos ou grupos. Dá pra controlar, por exemplo:

  • Quem pode ler, escrever ou executar um arquivo.
  • Quem é o dono do arquivo ou pasta.
  • Quais usuários têm permissões herdadas (recebem as mesmas permissões de uma pasta principal).

    • Essas permissões são gerenciadas por uma interface gráfica, onde o usuário clica com o botão direito no arquivo > “Propriedades” > “Segurança”.

Como funciona no Linux (sistema aberto – rwx)

No Linux, as permissões são baseadas em letras e divididas em três grupos:

  • r = read (ler)
  • w = write (escrever)
  • x = execute (executar)

    • No Linux, isso é feito normalmente via linha de comando, usando comandos como chmod, chown e ls -l.
